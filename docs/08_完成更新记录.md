## 如何追加

| 字段      | 说明                                        |
| --------- | ------------------------------------------- |
| 日期      | `YYYY-MM-DD`                                |
| 标题      | 一句话概括                                  |
| 类型      | 阶段验收 / 增强 / 体验 / 修复 / 文档        |
| 范围      | 模块或端（Windows / Android / protocol 等） |
| 要点      | 3–8 条用户可感知或架构级变化                |
| 验收/测试 | 如 `pytest` 结果、手工路径                  |
| 关联      | schema/协议/提交（可选）                    |

## 更新日志

### 2026-07-22 · Windows 主界面现代化改版

- **类型：** 体验
- **范围：** `apps/windows` UI（`theme.py` / `widgets.py` / `main_window.py` / `settings_dialog.py` / `__main__.py`）
- **要点：**
  - 左侧主导航：题库 / 复习 / AI / 数据 / 设置（`QStackedWidget`）
  - 题库页飞书式三栏：筛选 · 列表 · 属性；选中后才显示上下文操作条
  - 顶部动作墙收纳为「主按钮 + 更多菜单」；复习/AI/数据页卡片化聚焦
  - 全局浅色蓝白 QSS（主色 `#3370FF`）；设置对话框卡片分组
- **验收/测试：** `pytest tests/unit -q` → 52 passed；手工启动 `python -m yancuo_win`
- **关联：** 业务槽不改协议；无 schema 变更

### 2026-07-22 · AI 密钥可在设置写入系统凭据

- **类型：** 增强
- **范围：** AI 提供商、配置、设置页、单测
- **要点：**
  - `AiProviderConfig.credential_key`（默认 `yancuo_ai_api_key`）
  - 读取顺序：环境变量 `YANCUO_AI_API_KEY` → keyring
  - 设置中可保存/清除 AI 密钥并「应用 AI 到当前会话」
  - 默认视觉模型示例对齐 Faro：`gpt-5.6-sol`；`base_url=https://faroapi.com/v1`
- **验收/测试：** `test_ai_credentials.py` + 配置加载单测
- **关联：** 提交 `9938d7f`
