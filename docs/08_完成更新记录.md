- **类型：** 增强 / AI 搜索 / 隐私与安全

  - AI 搜索使用“SearchSpec 解析 → 本地有限召回 → 候选 JSONL 重排”的两阶段链路
  - 关键词 all/any、结构化筛选、知识目录和生命周期状态均在本地执行，AI 不能扩大范围

  - OpenAI 兼容提供商支持真实结构化文本请求，网络或响应错误不会触发不安全回退
  - 后端保留模型、token、费用和重试元数据，供下一切片诊断与界面异步接入
- **兼容边界：** 不升级数据库 schema 或跨端格式；题库 AI 搜索按钮仍待 `SEARCH-06` 完成诊断和后台执行后开放
